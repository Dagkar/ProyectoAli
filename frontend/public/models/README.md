# Modelos 3D de Pedales

Esta carpeta contiene los archivos de modelos 3D en formato `.glb` para cada pedal.

## Archivos necesarios:

- `pedal-tremolo.glb` - Modelo 3D del Pedal Trémolo
- `pedal-phaser.glb` - Modelo 3D del Pedal Phaser
- `pedal-flanger.glb` - Modelo 3D del Pedal Flanger
- `pedal-pitchshifter.glb` - Modelo 3D del Pedal Pitch Shifter
- `pedal-noisegate.glb` - Modelo 3D del Pedal Noise Gate
- `pedal-whammy.glb` - Modelo 3D del Pedal Whammy
- `pedal-vibrato.glb` - Modelo 3D del Pedal Vibrato

## Formato de archivo:

Todos los archivos deben ser en formato **glTF 2.0 con extensión .glb** (binario).

## Cómo obtener los modelos:

1. **Crear desde cero**: Usar Blender o similar para modelar los pedales
2. **Descargar**: Buscar en sitios como Sketchfab, TurboSquid, etc.
3. **Usar modelos de prueba**: Mientras obtienes los modelos finales, puedes usar modelos genéricos

## Características recomendadas del modelo:

- Resolución optimizada (10K-50K polígonos)
- Texturas PBR incluidas
- Escala correcta
- Orientación apropiada
- Tamaño de archivo: < 10MB

## Nota sobre Model Viewer:

Si los modelos no cargan, verifica:
- Que el archivo `.glb` esté en la carpeta correcta
- Que el nombre del archivo coincida exactamente en el HTML
- Que el servidor esté sirviendo correctamente los archivos CORS
